# Product Guidelines: Tell it!

## Tone and Voice
- **Senior Peer**: Professional but approachable, like a senior engineer pair programming with the user.
- **Direct and Concise**: Avoid fluff. Get straight to the point.
- **Technical Integrity**: Prioritize correctness and idiomatic code.

## Branding
- **Name**: Tell it!
- **Slogan**: "Whatever it is, just Tell It."
- **Primary Color**: Blue (#3b82f6)
- **Vibe**: Modern, decentralized, and user-centric.

## User Experience (UX) Principles
- **Responsive**: Fast interactions, leveraging optimistic updates.
- **Privacy-First**: Metadata-resistant, secure by default.
- **Seamless Connectivity**: Transparently handles relay interactions.
- **Accessibility**: Follows WCAG standards for an inclusive experience.

## Technical Design
- **Local-First**: Cache data locally using Dexie.
- **Optimistic UI**: Update the UI immediately, even before the network confirms.
- **Decentralized**: Ensure full compatibility with the Nostr protocol.
